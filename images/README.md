# ☕ Java Jam Coffee Bar – Product Page

## 📌 Overview

This project is a front-end recreation of a coffee shop product page based on a provided mockup. The goal was to practice HTML structure, CSS layout techniques, and visual styling to closely match a real-world design.

---

## 🎯 Objectives

- Recreate a webpage from a static design reference
- Practice semantic HTML structure
- Use CSS for layout, spacing, and styling
- Implement responsive design basics

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- Google Fonts (Pacifico)

---

## 🧩 Features

- Custom header with logo and styled title
- Navigation bar
- Sidebar filters (radio buttons + checkboxes)
- Product grid layout
- Styled product cards with pricing and actions
- Advertisement column
- Informational footer sections
- Responsive behavior for smaller screens
